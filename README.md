# 宠物医院管理系统 java964

## 项目概述

宠物医院管理系统是基于Spring Boot的Java项目，采用SSM框架进行开发。本项目旨在提供一套完善的宠物医疗管理系统，涵盖用户管理、宠物信息管理、预约管理和日常健康指导等多个功能模块。

## 技术栈

- **后端：**
  - Spring Boot
  - MyBatis
  - Spring
  - Spring MVC
- **前端：**
  - HTML
  - CSS
  - JavaScript
- **存储：**
  - MySQL
- **项目管理工具：**
  - Maven

## 功能模块

### 系统管理模块

- 用户管理
- 页面管理
- 角色管理

### 宠物管理模块

- 宠物列表
- 宠物状态
- 宠物健康史

### 预约管理模块

- 预约列表
- 预约状态
- 医生时间管理

### 日常健康模块

- 健康指南
- 健康标准
- 宠物日志

## 系统角色

系统设定了以下三个角色：

- 宠物医生
- 宠物主人
- 系统管理员

## 运行环境

- Java环境
- MySQL数据库
- Maven构建工具

## 部署步骤

1. 环境准备：确保Java和MySQL环境已正确安装。
2. 数据库部署：执行SQL脚本，建立数据库及数据表。
3. 项目构建：使用Maven进行项目构建。
4. 配置修改：根据实际环境修改配置文件。
5. 项目启动：运行Spring Boot项目。
